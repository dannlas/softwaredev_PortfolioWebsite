Projects

This portfolio website was created using HTML, CSS and Javascript and links to my web applications as well as provides some information about me, my experience and contact details. Most other projects take use of at least HTML and CSS where necessary.

My number decipher project uses Javscript to extract a number from a text describing the number in words. Text doesn't necessarily have to be perfect as spaces and other text noise is accounted for.

The converted result is both a clean version of the text as well as the number as an integer digit. Colour-coding can be turned on to more clearly see how the result is organised.

Another project I created with Javascript functionality is my panel builder project. The idea behind it is to allow you to add, edit, remove and move elements as well as panels themselves which contain the elements.

This project is a work in progress and more experimental of what can be achieved with Javascript. Making this project made me realise I had been creating something akin to that of a website builder.

The most technically complex project I have created so far is my pathding algorithm project, which I made using Python to handle the computational logic. In addition, I used PyScript for interpreting the code and making sure it could run as a web application.

The grid can be customised by clicking on a cell. Each cell will either be blocked "I", traversible "O", starting point "S" or end point "X". Pressing the find button will display the route the algorithm has taken to get from the start to the end point. If a path is unavailable, the algorithm will search available area from the start point until it determines that no route to the end point is possible. NOTE: the outer cells are set as walls for the algorithm to not treat the gaps as a wrap-around route due to the way the cell generation is integrated.

For my SQL project, I created a file for setting up a database as well as the tables and examples of retreiving data from them. The table information stored is a addendum stretch goal to my Python pathfinding algorithm for retrieving grid cells stored as strings so that they could later be loaded into the grid layout.
